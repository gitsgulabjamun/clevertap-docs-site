# Get Back Non-Returning Users

Use our “Get back non-returning users” campaign to tap new users on Day 20 (or any number of days you select) with a push notification that drives them back into your app if they haven’t returned to your app in that number of days.

# Send App Version Upgrade Reminders

Use our “App version upgrade reminders” campaign to remind users to download the new version of your app with a push notification and an email. This will happen automatically when CleverTap recognizes that you’ve updated a version of your app.
